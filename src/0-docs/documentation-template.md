# Documentation template

Below is a basic template for scss and the structure of it documentation

```scss
// Section title
//
// Section description here
//
// Markup:
// <div class="example-class">
//  Example markup
// </div>
//
// Weight: 2
//
// Styleguide: Folder.Path.Name

$variables: here;

@mixin here() {

}

.class {
  @include here;
}
```
